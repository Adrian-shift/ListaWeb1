Scripts Avançados de ANSI no Terminal

Além de mudar as cores do texto e do fundo, os códigos ANSI permitem criar interfaces ricas no terminal manipulando a posição do cursor e apagando conteúdos dinamicamente.

Mover o Cursor [cite: 14]

Você pode mover o cursor para qualquer direção usando as sequências abaixo:

Código ANSI Função
\033[FMove para cima uma linha [cite: 15]
\033[EMove para baixo uma linha [cite: 15]
\033[<N>AMove N linhas para cima [cite: 15]
\033[<N>BMove N linhas para baixo [cite: 15]
\033[<N>CMove N colunas para a direita [cite: 15]
\033[<N>DMove N colunas para a esquerda [cite: 15]
\033[<linha>;<coluna>HMove para uma posição específica (coordenadas) [cite: 15]

Salvar e Restaurar a Posição do Cursor [cite: 18]

Código ANSIFunção
\033[sSalva a posição atual do cursor [cite: 19]
\033[uRetorna a última posição salva do cursor [cite: 19]

Apagar Linhas e Caracteres [cite: 16]

Ideal para criar barras de carregamento ou limpar telas de menus:

Código ANSI Função
\033[KApaga do cursor até o final da linha [cite: 17]
\033[1KApaga do cursor até o início da linha [cite: 17]
\033[2KApaga a linha inteira [cite: 17]
\033[JApaga do cursor até o final da tela [cite: 17]
\033[1JApaga do cursor até o início da tela [cite: 17]
\033[2JApaga a tela inteira [cite: 17]

Cores e Estilos Extras [cite: 26]

O formato padrão para combinar configurações é: \033[ESTILO; COR_TEXTO; COR_FUNDOm. Os comandos podem ser combinados usando ponto e vírgula (;)[cite: 30, 31].

Lista de Estilos

  • Negrito: \033[1m [cite: 27]
  • Sublinhado: \033[4m [cite: 28]
  • Pisca: \033[5m (Nem sempre funciona) [cite: 29]
  • Redefinir (Volta ao normal): \033[0m [cite: 29]

Observações e Compatibilidade [cite: 20]

Sistemas Suportados: Funciona no CMD, PowerShell, Linux Terminal e Mac Terminal (desde que suportem ANSI)[cite: 21].

Configuração no Windows: No CMD do Windows, pode ser necessário ativar o suporte ANSI[cite: 22]. Para isso, execute o comando abaixo no terminal:

reg add HKCU\Console /v Virtual TerminalLevel /t REG_DWORD /d 1 /f [cite: 23]

Para garantir compatibilidade de forma nativa no Windows sem alterar registros, pode-se usar a biblioteca windows.h e a função SetConsoleCursorPosition() em seu código C[cite: 24].
