Scripts Avançados de ANSI no Terminal

Além de mudar as cores do texto e do fundo, os códigos ANSI permitem criar interfaces ricas no terminal manipulando a posição do cursor e apagando conteúdos dinamicamente.

Mover o Cursor [cite: 14]

Você pode mover o cursor para qualquer direção usando as sequências abaixo:

Código ANSI Função
\033[FMove para cima uma linha
\033[EMove para baixo uma linha
\033[<N>AMove N linhas para cima
\033[<N>BMove N linhas para baixo
\033[<N>CMove N colunas para a direita
\033[<N>DMove N colunas para a esquerda
\033[<linha>;<coluna>HMove para uma posição específica (coordenadas)

Salvar e Restaurar a Posição do Cursor [cite: 18]

Código ANSIFunção
\033[sSalva a posição atual do cursor
\033[uRetorna a última posição salva do cursor

Apagar Linhas e Caracteres

Ideal para criar barras de carregamento ou limpar telas de menus:

Código ANSI Função
\033[KApaga do cursor até o final da linha
\033[1KApaga do cursor até o início da linha
\033[2KApaga a linha inteira
\033[JApaga do cursor até o final da tela
\033[1JApaga do cursor até o início da tela
\033[2JApaga a tela inteira

Cores e Estilos Extras

O formato padrão para combinar configurações é: \033[ESTILO; COR_TEXTO; COR_FUNDOm. Os comandos podem ser combinados usando ponto e vírgula (;).

Lista de Estilos

  • Negrito: \033[1m
  • Sublinhado: \033[4m
  • Pisca: \033[5m (Nem sempre funciona)
  • Redefinir (Volta ao normal): \033[0m

Observações e Compatibilidade

Sistemas Suportados: Funciona no CMD, PowerShell, Linux Terminal e Mac Terminal (desde que suportem ANSI).

Configuração no Windows: No CMD do Windows, pode ser necessário ativar o suporte ANSI. Para isso, execute o comando abaixo no terminal:

reg add HKCU\Console /v Virtual TerminalLevel /t REG_DWORD /d 1 /f

Para garantir compatibilidade de forma nativa no Windows sem alterar registros, pode-se usar a biblioteca windows.h e a função SetConsoleCursorPosition() em seu código C.
